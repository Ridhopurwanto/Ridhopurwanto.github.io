Ini adalah web sederhana untuk tes kecepatan menghitung dengan menentukan apakah hasil dari penjumlahan 2 bilangan merupakan bilangan ganjil atau genap. Tes dilakukan dengan memberikan beberapa soal dengan durasi tertentu untuk setiap sectionnya. Jumlah soal, durasi, dan jumlah section bisa diatur pengguna di menu awal sebelum memulai tes. Masih banyak kekurangan dari web ini salah satunya adalah tampilan web yang kurang responsif, tidak sesuai untuk ditampilkan di handphone.
